
Code Documentation Navigator
Practical Application

A zip file named Code_navigator.zip is provided which contains:

app.py

parser.py

README.py

Setup Instructions:

[Code_navigator.zip](https://github.com/user-attachments/files/25013376/Code_navigator.zip)

Save the zip file on your system and extract it.

Ensure Python 3.11.9 (64-bit) is installed and the path is defined.

Steps to Use Code Navigator:

Open Windows PowerShell or Command Prompt.

Navigate to the extracted folder:

cd C:\Code_navigator


Run the Streamlit app:

streamlit run app.py


A local host page will open in your browser.

Upload your Python code using the interface.

Click on Analyze Code.

The analyzed and navigated Python code will be displayed!
